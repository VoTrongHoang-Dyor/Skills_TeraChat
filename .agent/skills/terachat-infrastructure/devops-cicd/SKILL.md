# TeraChat Infrastructure DevOps & CI/CD Skill

Skill này quản lý quy trình tự động hóa việc build, test, và deploy ứng dụng TeraChat. Đảm bảo quy trình phát triển mượt mà và an toàn.

## Chức Năng Chính
- **CI/CD Pipeline**: Thiết lập và bảo trì GitHub Actions, Jenkins, etc.
- **Code Signing**: Quản lý certificate và process ký ứng dụng cho macOS/Windows.
- **Docker/Container**: Quản lý môi trường build và runtime container.
- **Release Automation**: Tự động hóa việc release version mới.

## Khi Nào Sử Dụng?
- Khi setup hoặc sửa lỗi pipeline build.
- Khi config environment variables hoặc secrets mới.
- Khi chuẩn bị release bản build production.
