# TeraChat Embedded IoT Engineering Skill

Skill này dành riêng cho việc phát triển các tính năng liên quan đến phần cứng (Hardware) hoặc thiết bị nhúng (Embedded Systems) nếu có trong tương lai.

## Chức Năng Chính
- **Firmware Development**: Viết firmware cho thiết bị IoT kết nối với TeraChat.
- **Hardware Interface**: Giao tiếp với các thiết bị vật lý.
- **Protocol**: MQTT, CoAP, BLE nếu cần.

## Khi Nào Sử Dụng?
- Khi TeraChat mở rộng sang các thiết bị phần cứng chuyên dụng.
- Khi cần test kết nối với thiết bị ngoại vi.
