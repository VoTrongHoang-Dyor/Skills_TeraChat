# TeraChat Infrastructure SecOps & Audit Skill

Skill này tập trung vào khía cạnh bảo mật vận hành và kiểm toán hệ thống. Đảm bảo mã nguồn và quy trình tuân thủ các chuẩn bảo mật cao nhất.

## Chức Năng Chính
- **Vulnerability Scanning**: Quét lỗ hổng dependency và code (SAST/DAST).
- **Security Audit**: Review cấu hình hạ tầng và quy trình deploy.
- **Compliance Check**: Đảm bảo tuân thủ các quy tắc bảo mật của dự án (ví dụ: không hardcode secrets).
- **Incident Response**: Hỗ trợ xử lý sự cố bảo mật.

## Khi Nào Sử Dụng?
- Khi cần audit code trước khi merge.
- Khi phát hiện lỗ hổng bảo mật mới (CVE).
- Khi review cấu hình security của hệ thống.
