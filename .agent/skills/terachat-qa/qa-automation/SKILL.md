# TeraChat QA Automation Skill

Skill này chịu trách nhiệm về tự động hóa kiểm thử (Test Automation) để đảm bảo chất lượng phần mềm.

## Chức Năng Chính
- **Test Scripting**: Viết script test tự động (Selenium, Playwright, or Unit Test Frameworks).
- **Regression Testing**: Chạy test hồi quy khi có thay đổi code mới.
- **Performance Testing**: Kiểm tra hiệu năng và khả năng chịu tải.
- **Bug Reporting**: Báo cáo lỗi một cách chi tiết và có hệ thống.

## Khi Nào Sử Dụng?
- Khi viết unit tests hoặc integration tests mới.
- Khi cần kiểm tra tính ổn định của bản build.
- Khi verify bug fix.
